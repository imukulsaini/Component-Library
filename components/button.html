Button

With support for multiple sizes, states, and more, use CRUX UI's custom button styles for actions in types, dialogues, and more.

Outline buttons

Want a button but don't want to deal with the hefty background colours that come with them? To delete all background images and colours from any button, replace the default modifier classes with the .btn-outline-* ones.

Button with Size

Add .btn-lg or .btn-sm to change button's size larger and smaller.
